# arduino-cj125
Arduino CJ125 library for O2 sensor
O2 5-wire sensor Bosch LSU4.9 wiring should be connected as follow:
        IP      red             (czerwony)
        VM      yellow          (żółty)
        HG      white           (biały)
        B+      grey            (szary)
        IA      not connected   (nie podłączony)
        UN      black           (czarny)
